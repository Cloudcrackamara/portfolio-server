Mediplus Healthcare Web App — End-to-End Cloud Deployment

A 3-tier healthcare application (Frontend, FastAPI Backend, Database) deployed using Docker, AWS ECR, EC2, and HAProxy. This project simulates a real-world healthcare infrastructure with full automation and secure configurations.

Tech Stack

Docker • AWS ECR • EC2 • HAProxy • GitHub Actions • Linux • Nginx

What I Did

  • Containerized both frontend and backend using optimized multi-stage Dockerfiles.
  • Built a CI/CD pipeline with GitHub Actions to automate Docker image builds and ECR pushes.
  • Configured EC2 instance and deployed containers using Portainer for easier management.
  • Set up HAProxy as a reverse proxy to route traffic securely and efficiently.
  • Implemented IAM roles, Security Groups, and HTTPS for enhanced application security.

Outcome

Deployment time was reduced and reliability improved through full automation and containerized application delivery.
